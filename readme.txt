== Description ==

This plugins adds a real, true and personal signature to your posts. This plugin allows you to easily use your free or animated live personal signature (from http://www.mylivesignature.com) on your blog.

The plugin is extremely easy to use: Just set your MyLiveSignature generated Signature ID (totally free), save, and from that moment on your personal signature will be automatically added below each post you make. Want to stop showing your signatures? Also easy - just disable the plugin.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to settings -> MyLiveSignature and set your signature Id value (get it free at mylivesignature.com).
4. Save and preview your signature. If it's blank - double check your signature Id.

== Frequently asked questions ==

= My Signature shows as blank

Make sure your signature Id is correct.
